Sobre mim

Transformando aprendizado em software de verdade.

Minha jornada começou com projetos acadêmicos e evoluiu para aplicações completas, organizadas e pensadas para resolver problemas reais.

Sou estudante de Sistemas de Informação e desenvolvedor focado no ecossistema Java. Gosto de entender como cada parte do sistema funciona, desde a modelagem do banco de dados até a experiência final de quem utiliza a aplicação.

Ao longo dos meus projetos, venho aprofundando conhecimentos em Java, Spring Boot, JavaFX, bancos de dados, segurança, arquitetura em camadas, Git e desenvolvimento web.

Meu objetivo é construir software com código limpo, organização, boas práticas e capacidade de evoluir.
